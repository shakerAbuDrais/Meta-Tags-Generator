# React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## AI Meta Tag Generator from Google Sheet

This application allows you to generate meta titles and descriptions for a list of URLs provided in a Google Sheet. It uses the OpenAI API to generate the meta tags based on the content fetched from each URL.

### Prerequisites

1.  **Node.js and npm/yarn:** Ensure you have Node.js (which includes npm) or yarn installed.
2.  **OpenAI API Key:** You need an API key from OpenAI.
3.  **Google Sheet:** A Google Sheet containing URLs in the first column (Column A). This sheet must be public or accessible via a link ("Anyone with the link can view").

### Setup

1.  **Clone the repository (if applicable) or download the files.**
2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```
3.  **Create an environment file:**
    Create a `.env` file in the root of the project directory.
    Add your OpenAI API key to this file:
    ```env
    VITE_OPENAI_API_KEY=your_openai_api_key_here
    ```
    Replace `your_openai_api_key_here` with your actual OpenAI API key.

### Running the Application

1.  **Start the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```
2.  Open your browser and navigate to the URL provided by Vite (usually `http://localhost:5173` or similar).

### How to Use

1.  **Prepare your Google Sheet:**
    *   Ensure the first column (Column A) contains the list of full URLs (e.g., `https://example.com/page1`).
    *   Make your Google Sheet public:
        *   Click "Share" in your Google Sheet.
        *   Under "General access," change to "Anyone with the link."
        *   Ensure the permission is set to "Viewer."
        *   Copy the share link.
2.  **Get the CSV Export Link (Recommended):**
    While the application attempts to parse standard Google Sheet URLs, providing a direct CSV export link is more reliable.
    *   You can construct this from your sheet's normal URL:
        If your sheet URL is `https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/edit#gid=YOUR_GID`
        The CSV export link is `https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/export?format=csv&gid=YOUR_GID`
    *   If it's the first sheet (gid=0), the `&gid=YOUR_GID` part might be optional or `&gid=0`.
3.  **Enter the Google Sheet URL:**
    *   In the application, paste either your standard Google Sheet share link or the direct CSV export link into the input field.
4.  **Generate Meta Tags:**
    *   Click the "Generate Meta Tags" button.
    *   The application will fetch the URLs from your sheet, retrieve content from each URL (using a CORS proxy), and then call the OpenAI API to generate a meta title and description.
    *   Results will appear in the table below the input field, showing the original URL, generated title, generated description, and the status of each operation.

### Notes

*   **CORS Proxy:** The application uses a public CORS proxy (`https://api.allorigins.win/`) to fetch content from URLs. The reliability of this proxy can vary, and some websites might still block it. For more robust fetching, a dedicated backend proxy might be required.
*   **OpenAI API Usage:** Meta tag generation consumes OpenAI API tokens. Be mindful of your usage and quotas.
*   **Content Extraction:** The text content extracted from URLs is a simplified process (stripping HTML tags). The quality of extracted text can affect the meta tags generated by OpenAI.
*   **Error Handling:** The application displays errors if it encounters issues like invalid sheet URLs, failed content fetching, or OpenAI API problems. Check the status column in the results table for details.
